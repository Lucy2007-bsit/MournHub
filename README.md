# MournHub 👋

A random website created by an IT student for fun, featuring various sections like videos, games, and products.

## 🚀 Project Overview

MournHub is a personal project developed by Richard, an IT student. It's a collection of web pages designed for entertainment and showcasing basic web development skills. The website includes sections for videos, a simple minigame, and product displays, all themed around a playful interpretation of "mourning".

## ✨ Features

*   **Interactive Navigation:** A fixed top navigation bar allows easy switching between sections: Home, Videos, Games, Products, and About.
*   **Multimedia Content:** Integrates audio playback and video displays.
*   **Engaging Visuals:** Utilizes CSS animations for background gradients and element transitions, alongside Bootstrap for styling.
*   **Minigame:** A playable "Plants vs Crazy" game implemented using HTML Canvas.
*   **Product Showcase:** Displays various "Afterlife Beds" with pricing and a "Buy Now" button.
*   **Responsive Design:** Basic responsiveness with Bootstrap integration.
*   **Cursor Trail Effect:** A JavaScript effect that creates a trail of orange dots following the cursor.
*   **Scroll Animations:** Elements fade in as the user scrolls down the page.

## 🛠️ Tech Stack

*   **Languages:** HTML, CSS, JavaScript, Markdown
*   **Frameworks/Libraries:** Bootstrap 5.3.8, Font Awesome
*   **Other:** HTML Canvas (for the minigame)

## 📥 Installation

As this is a front-end project with no server-side components or external dependencies beyond CDN links, there are no specific installation steps required. You can simply clone the repository and open the `index.html` file in your web browser to view the website.

```bash
git clone https://github.com/Lucy2007-bsit/MournHub.git
cd MournHub
# Open index.html in your web browser
```

## ▶️ Usage

Navigate through the website using the top navigation bar.

*   **Home:** The landing page with introductory information and links.
*   **Videos:** A collection of embedded video content.
*   **Games:** Access to the "Plants vs Crazy" minigame.
*   **Products:** A showcase of "Afterlife Beds" with pricing.
*   **About:** Information about the website and its creator.

### Play the Minigame 🎮

1.  Navigate to the "Games" section.
2.  Click the "Play PVC Game 🌱" button.
3.  Follow the on-screen instructions to place plants and defend against zombies.

### Exploring Videos 🎬

Visit the "Videos" page to watch a selection of meme-related video clips.

### Viewing Products 🛏️

Go to the "Products" page to see a catalog of "Afterlife Beds" with their respective prices.
